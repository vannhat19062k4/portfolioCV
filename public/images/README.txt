Drop your images here for the site to use.

Required filename for default hero: me.jpg
Recommended size: 1000x1200 (or at least 500x600) — image will be resized responsively.
Supported formats: .jpg, .jpeg, .png, .webp

Usage:
- Hero / profile image (used by default): /images/me.jpg
- You can also reference images from components via '/images/<filename>'

After adding images, reload the dev server or refresh the page to see updates.
